# Calculadora Inteligente EBAC
Calculadora simples criada durante as aulas para realizar contas básicas (Soma, Subtração, Divisão e Multiplicação) com números reais.

Recursos necessários: 
* Terminal Ubuntu 22.04
* GitHub

Para executar o arquivo sh. é necessário que abra o seu terminal, coloque-o no seu diretório e em seguida digite os seguintes comandos:

```
./calculadora_script.sh
```
Após realizar isso, o script irá instalar os recursos necessários para executar o arquivo em python.
